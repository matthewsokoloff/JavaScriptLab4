Page 3: DataTypes

JavaScript has seven different data types:

  • string
  • number
  • boolean
  • object
  • function
  • null
  • undefined

You can see the data type for something using the typeof() command

For example, to log the data type of "Hello" to the console, you would type console.log(typeof("Hello"))
